# Passo a passo para configurar VS Code

---

## 1. Instalar VS Code
- Baixe no site oficial: **Visual Studio Code - The open source AI code editor**

---

## 2. Instalar extensões essenciais
- **Python (Microsoft):** suporte completo para Python.
- **Pylance:** autocompletar e análise de código.
- **Jupyter:** rodar notebooks dentro do VS Code.
- **Black Formatter:** mantém o código sempre bem formatado.
- **isort:** organiza automaticamente os imports.
- **GitHub Pull Requests and Issues:** integração direta com GitHub.
- **GitLens:** facilita visualizar histórico e autoria de código.

---

## 3. Inicializar o projeto com Poetry
No Git Bash dentro da pasta do seu projeto, rode:

```bash
poetry init
